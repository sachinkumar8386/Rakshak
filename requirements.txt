# Requirements for Rakshak

## System Requirements

- Windows 10/11 (64-bit)
- At least 4GB RAM
- 100MB free disk space

## Software Requirements

### For Development

- **Rust**: Latest stable version (via rustup)
- **Node.js**: v18 or higher
- **npm**: v9 or higher
- **Visual Studio Build Tools 2022** (Windows)
